This project analyzes Diwali festival sales data to uncover insights about customer purchasing behavior, product performance, and regional sales trends.

The dataset (11,000+ rows) was cleaned and prepared using Excel, and visualized in Power BI through an interactive dashboard.

## Key Features

Data cleaning and preprocessing in Excel

KPI creation (Total Sales, Orders, Customers, AOV, Revenue per Customer)

DAX measures and calculated columns

Customer segmentation (age group, spending level, new vs repeat)

Interactive visuals and filters

## Insights Generated

Identified top-performing states and cities

Analyzed gender and age-based spending patterns

Found high-revenue product categories

Evaluated customer loyalty and repeat purchases
